SetCreator

SetCreator(string creator [, boolean isUTF8])

Descripción

Define el creador de el documento. Este es típicamente el nombre de la aplicación que genera el pdf.

Parámetros

creator
El nombre del creador.
isUTF8
Indica si la cadena está codificada en la norma ISO-8859-1 (false) o UTF-8 (true).
Valor por defecto: false.
